# 🔬 Peer Review Verification System
A decentralized platform for scientific peer review management and verification on the Stacks blockchain.

## 🚀 Features

- Scientists can register and submit research
- Qualified reviewers can submit peer reviews
- On-chain verification of review process
- Reputation system for both scientists and reviewers
- IPFS integration for research storage

## 📋 Requirements

- Clarinet
- Node.js
- Stacks Wallet

## 🛠️ Setup

```bash
clarinet new peer-review-verification-system
cd peer-review-verification-system
```

## 🧪 Testing

```bash
clarinet test
```

## 🔒 Security Features

- Owner-only administrative functions
- Input validation for scores and status
- Double-review prevention
- Qualification checks for reviewers

## 💻 UI Components

- Research submission dashboard
- Reviewer portal with pending reviews
- Research status tracking interface
- Reputation score visualization
